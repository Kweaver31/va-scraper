  va-webscraper is a program that will scan https://www.va.gov/ get all necessary information about Veteran benefits
  The program will not run without installing the requests, and beautifulsoup4 libraries
  Assuming you have Python3 and pip3 installed both libraries can be installed by running the command 'install.sh' from the va-webscraper current directory
  To start va-webscraper, open terminal and run the command 'python3 va-webscraper.py' from the va-webscraper current directory

  Program won't run without intalling libraries below (Covered by install.sh)
  pip3 install requests
  pip3 install beautifulsoup4
